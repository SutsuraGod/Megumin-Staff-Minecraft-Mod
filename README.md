# Megumin Staff
This mod adds magic staff from Konosuba anime.

## Installation Guide
If you do it for the first time then read this guide.

Firstly, download Fabric Loader from their official website https://fabricmc.net/use/installer/. Run it and choose **26.1.2** Minecraft version and **0.19.2** Fabric Loader version. It can suggest you download Fabric API, do it from their github repository https://github.com/FabricMC/fabric-api/releases.

Secondly, download the latest release of my mod from https://github.com/SutsuraGod/Megumin-Staff-Minecraft-Mod/releases.

After you downloaded this mod and Fabric API, put them into your **.minecraft/mods** folder. Then run Minecraft and enjoy the game!
